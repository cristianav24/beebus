Attendance with QR (Flutter + Laravel)

Introduction

How to get started with Attendance with QR (Flutter + Laravel)!

Thank you for downloading our system! We really appreciate it and ready to help you!

This guide will help you get started with the system! This includes starting with the system, how to use, customization, and if there are bugs or questions please contact us. Please follow the instructions carefully. So let's get started!



Setup System (Laravel)

1. Extract the zip file and place it in a public folder on your hosting

2. Import the SQL file (root folder -> sql_file.sql).

3. If you use the vendor folder please type on your command prompt (composer update).


composer update

If not please delete the vendor folder, and run (composer install).


composer install

4. Clear cache, config, view.


php artisan cache:clear
php artisan config:clear
php artisan view:clear

5. Do settings

Settings Database and email go to (.env).


DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=YOUR DATABASE
DB_USERNAME=YOUR USERNAME
DB_PASSWORD=YOUR PASSWORD

MAIL_DRIVER=smtp
MAIL_HOST=smtp.mailtrap.io
MAIL_PORT=2525
MAIL_USERNAME=YOUR USERNAME
MAIL_PASSWORD=YOUR PASSWORD
MAIL_FROM_ADDRESS=YOUR EMAIL
MAIL_FROM_NAME="${APP_NAME}"

MAIL_FROM_ADDRESS=YOUR EMAIL FROM
MAIL_FROM_NAME="${APP_NAME}"

6. Try to visit your website (http://Your Domain Name/public/login).


Login details
User: administrator@gmail.com
Password: 123456789

User: admin@gmail.com
Password: 123456789

User: staff@gmail.com
Password: 123456789

7. Done



Setup Application (Flutter)

1. For first-time applicants need to add configuration KEY and URL, you need to find the QR code from system management,(system management -> settings)

Next, Scan the QR code via the app

2. To make attendance, log in first, and then go to(Dashboard -> History QR) make sure you already generate the QR code for your employees.

*Note: You can save or print the QR code for your employee or student.

3. Scan the QR code.

 

4. Done.

Build the Android project from Android Studio or another software. You can testing with your Android smartphone, or build the app-debug.apk and install it on your Android smartphone. For more info visit this Build and run your app.

Build the IOS project from Xcode. You can testing with your IOS smartphone. For more info visit this Build and run your app.

*Note: we have included the APK file in the zip, for IOS users please test with Xcode.

How to import from CSV?

1. Visit your website where the system has been installed

2. (Login to your page)

3.(Go to -> History QR -> Import CSV)

4. Please download first the CSV template

5. To add data please follow the instructions by clicking the "Show Instructions"

6. upload the CSV file

7. Click import

8. Done

*Important: Please pay attention, this system has 2 types of users. The first user with login means this user can attend and also login to system web, the second user without login means this user only attend and cannot login to the system. This CSV only work with role 2 and 3 (admin, and staff).

How to change the application launcher icon on (Flutter)?

1. Copy your icon to this path (assets/icon/icon.png). You can change the name of your icon from icon.png to your icon name.

2. Go to terminal and execute this command


flutter pub get
flutter pub run flutter_launcher_icons:main

3. Done



How to change package name and app name (Flutter)?

1. Change the package name

Go to the terminal and execute this command


flutter pub run change_app_package_name:main com.new.package.name

Where (com.new.package.name)is the new package name that you want for your app. replace it with any name you want.


2. Change the name of the app

By default, when a flutter app gets installed, the app name on the launcher is your Flutter project name. To change that to your desired application name on Android or iOS both, you need to change AndroidManifest.xml and Info.plist respectively.

Android

Inside, find <application> and change its android:label property with your desired app name.

(android/app/src/main/AndroidManifest.xml)


<application
        android:name="io.flutter.app.FlutterApplication"
        android:label="your_app_name"
        android:icon="@mipmap/ic_launcher">

IOS

(ios/Runner/Info.plist)


<key>CFBundleName</key>
<string>your_app_name</string>

How to change the Favicon?

1. Generate your favicon first. You can generate the favicon from many tools, e.g https://www.favicon-generator.org/

2. Go to Laravel path.

3. (root/public/favicons) replace the favicon with your favicon to that folder.

4. In your Browser click Hard Reload, or use incognito to check your new favicon.

5. Done
